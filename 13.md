# Light
## Features
- Supports Brightness
- Supports Color Temperature
- Supports Effect (Scene)

## Customize

Set the light's color temperature range in Kelvin ([2700, 6500] by default).

```json
{"color_temp_range_kelvin": [2000, 6800]}
```
